# 崩坏3 说书神器

## 完全开源，用 AutoHotkey 语言写的 Windows 软件，用以在游戏崩坏3聊天频道里更方便快捷的说书。

### 软件处于测试阶段，还不稳定，请注意及时备份，必免丢失！!

### 后续将陆续添加完善功能

### 欢迎大家提供更多需要和谐的词语，格式：和谐词语=需转换的词
